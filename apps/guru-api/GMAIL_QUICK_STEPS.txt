GMAIL EMAIL SETUP - QUICK STEPS
================================

1. Enable 2FA:
   https://myaccount.google.com/security
   → 2-Step Verification → Turn on

2. Generate App Password:
   https://myaccount.google.com/apppasswords
   → Select "Mail" → "Other (Custom name)" → "Guru API"
   → Copy 16-character password (remove spaces)

3. Add to .env:
   SMTP_USER=your-email@gmail.com
   SMTP_PASS=your-app-password
   SMTP_HOST=smtp.gmail.com
   SMTP_PORT=587

4. Test:
   python3 -c "from src.notifications.channels.emailer import send_email; print(send_email('test@example.com', 'Test', 'Hello'))"
